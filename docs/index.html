SPBChurch Radio

Android Kotlin Jetpack Compose MIT

Android-приложение для прослушивания онлайн-радио и каталога аудиозаписей церкви ЕХБ «Преображение» (Санкт-Петербург).

Возможности

🎙️ Прямой эфир

  • Стриминг радиопотока в реальном времени
  • Отображение текущего играющего трека
  • Индикатор "ЭФИР" с пульсирующей точкой
  • Буферизация для стабильного воспроизведения

🎵 Библиотека треков

  • Каталог из 2000+ аудиозаписей
  • Поиск по названию
  • Сортировка: по умолчанию, А–Я, Я–А

▶️ Воспроизведение

  • Микс — случайный порядок
  • Повтор — по кругу
  • До конца — один раз
  • Кнопки вперёд/назад, перемотка

📥 Офлайн-режим

Загрузка треков для прослушивания без интернета с отслеживанием прогресса.

❤️ Избранное

Собственный список любимых треков с быстрым добавлением и удалением.

🔔 Фоновое воспроизведение

Музыка играет при свёрнутом приложении с управлением с экрана блокировки.

Установка

Скачивание APK

  1. Перейдите на страницу Actions
  2. Выберите последний workflow run
  3. Скачайте artifact app-debug

Установка на Android

  1. Откройте скачанный APK
  2. Разрешите установку из неизвестных источников
  3. Нажмите "Установить"

Требования

  • Android 8.0 (API 26) или выше
  • ~50 МБ свободного места
  • Интернет-соединение

Разработка

Требования

  • Android Studio Hedgehog (2023.1.1)+
  • JDK 17+
  • Android SDK 34

Сборка

git clone https://github.com/Slakwik/spbchurch-radio-android.git
cd spbchurch-radio-android/android
./gradlew assembleDebug

FAQ

Это официальное приложение?

Да, приложение создано при поддержке церкви для удобства прихожан.

Сколько стоит?

Полностью бесплатно и без рекламы.

Можно ли слушать без интернета?

Да, для этого нужно предварительно скачать треки. Радиоэфир требует интернета.

Как сообщить о проблеме?

Создайте issue на GitHub.
