# Установка

## Установка через APK

### 1. Скачайте APK

**Вариант A — GitHub Releases:**
1. Перейдите на [страницу Releases](https://github.com/Slakwik/spbchurch-radio-android/releases)
2. Скачайте последний `app-release.apk`

**Вариант B — GitHub Actions:**
1. Перейдите в [Actions](https://github.com/Slakwik/spbchurch-radio-android/actions)
2. Выберите workflow run
3. Скачайте artifact `app-debug`

### 2. Разрешите установку из неизвестных источников

**Android 8.0+:**
1. Откройте скачанный APK
2. Система предложит разрешить установку
3. Нажмите "Настройки" → "Разрешить"

**Android 7 и ниже:**
1. Настройки → Безопасность
2. Включите "Неизвестные источники"
3. Или "Установка неизвестных приложений" → выберите браузер/файловый менеджер

### 3. Установите

1. Откройте APK файл
2. Нажмите "Установить"
3. Готово!

## Установка через Google Play (планируется)

Приложение планируется к публикации в Google Play.

## Требования

- Android 8.0 (API 26) или выше
- ~50 МБ свободного места
- Интернет-соединение (для радио и загрузки треков)
